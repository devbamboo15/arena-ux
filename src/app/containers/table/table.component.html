Custom ngx-datatable

aux-table is ngx-datatable wrapper allowing more unified form of displaying data from arena API. It unifies API's Meta and Request objects. It uses same API as ngx-datatable so refer to official ngx-datatable documentation.

IMPORTANT Only TemplateRef column templating is supported (to keep code clean).

Wrapped ngx-datatable supports all bootstrap styling classes like .table-sm or .table-hover so refer to official documentation for that.
