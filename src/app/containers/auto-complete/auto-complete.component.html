Autocomplete

Component to use autocomplete endpoints.

Installation

To use aux-autocomplete install it this way.

Usage

Provide apiUrl for direct API calls.

  • user name : {{items.name}}

    user uuid : {{items.user_uuid}}

  • user name : {{items.name}}

    user uuid : {{items.user_uuid}}

Inputs

# Param Description Type Default Available options
1 apiUrl API relative URL string none
2 queryParam Search query param for api endpoint string none
3 isMultiSelect Configure auto-complete mode. Multi-select on/off boolean true true, false
4 bindLabel Bind the label in object string 'name'
5 value Default Value Array none

Outputs

# Param Description
1 valueSelected Emit selected values in array. When multiSelect is false, array value with one element is emitted.
