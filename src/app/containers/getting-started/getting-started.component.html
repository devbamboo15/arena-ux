Getting started

@arenaops/arena.ux contains all core components and API services and storage powered by Angular and NGRX for Arena.

Dependencies

Exact same versions of those packages must be installed, to avoid any issues.

Installation

Install @arenaops/arena.ux from npm repository

Import package `module` in your `app.module.ts` additionally you will need to import `StoreModule` to have access to functionalities like `authorizations`, `support desk`, `notifications` or `layout states`.

Import arena.ux styles into your global styles

You will need to use core components to initialize the app the content of your `app.component.ts` should be.
